补充：怎么用这 10 题测 RAG 召回（建议）

1）设计查询方式

* 每题写成 1–2 条不同风格的 query：

  * 工程师口语版（中/英混合、带产品背景）
  * 条文检索版（含标准号 + 关键术语）
* 看不同 query 下 top-k 召回的条款是否稳定命中同一章节。

2）度量召回

* 对每题先人工标一个「必须召回的条款 ID / 页码范围」。
* 对每次 query，检查：

  * 是否在 top-k（例如 k=10）里出现这些 chunk；
  * 若没出现，分析是切块问题、embedding 问题，还是缺 metadata 过滤（标准号、产品类型）的问题。

3）分维度看问题

* 标示类问题（Q3、Q5、Q8、Q9、Q10）：考察对「label / marking / URL / QR code」相关词的词法匹配。
* 结构及材料类（Q1、Q5、Q6、Q7、Q10）：考察对「enclosure / polymeric / drain openings / field-wiring」等术语的语义理解。
* 试验方法类（Q2、Q4、Q8、Q9）：考察对「test / hose stream / SCCR / photobiological」等试验章节的召回。

你可以先只用这 10 题跑一轮 log，看看每题有无「完全 miss 正确章节」的情况，再决定是先调切分、索引策略，还是先调 embedding / rerank。

- 將燈光優化至專業攝影級品質：平衡曝光、柔和自然陰影、真實光衰減、細膩環境光遮蔽、一致的光照方向以強化形體層次與材質質感
- 若設計暗示存在發光面（如擴散罩、LED模組、燈頭），則將其呈現為真實光源，具有自然輝光、真實亮度、柔和光暈，以及合理的光線溢出，同時避免局部過曝

- 避免任何風格化、卡通化、繪畫感或明顯CGI效果：最終圖像必須呈現高端專業產品攝影效果，而非渲染或插圖
- 保持清晰形體邊緣，同時避免過度銳利的人工輪廓：應用自然邊緣柔化、適度景深，以及與原始焦距相符的真實鏡頭光學特性
- 采用基於物理的著色與全局光照：真實的間接光、顏色反射、陰影柔化，增強畫面深度與三維感
- 在最終圖像中移除所有技術標注（如尺寸線、引出線、標簽或CAD圖框信息），只保留完整實現的燈具與環境
- 進行專業且輕度的色彩調整：中性白平衡、準確材質色彩、適度對比度，無強烈濾鏡或藝術化色調
- 輸出幹凈、無噪點、高分辨率、可直接用於客戶展示的攝影級精修效果，無水印、標識、邊框或任何文字覆蓋
